About This Work

“A P5 Infinite Rogue” is a procedurally generated endless adventure game built entirely from scratch. It features completely randomly generated levels, buffs, and enemies. The randomly generated levels prevent players from getting bored, encouraging them to try different buffs to alter their gameplay experience. Interestingly, because the game has very little negative feedback, I have found many players attempt to speedrun each level. Using the P5 editor, this is the game with the most original code I have ever written, the first game I truly built entirely from scratch. Its code is approaching 1000 lines, making it an incredibly large and complex system for me at this stage. The code for randomly generating levels, enemies, and buff triggers constitutes the majority and has consumed most of my development time. I referenced the village generation system from Minecraft, where each asset has an entrance and an exit, and each level's entrance is linked to an exit, forming a linear level structure. I tagged each floor and defined the spawn rules and behaviors for enemies and triggers.

Keywords: game-design, complex-system, procedural-generation, random
