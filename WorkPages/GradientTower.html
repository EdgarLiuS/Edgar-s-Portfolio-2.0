Gradient Tower

A Procedurally Generated Text Poster with Gradient Effects

About This Work

This is one of my latest P5 projects. Using the TextToPoints function in P5.js, I was able to manipulate text effects with greater finer detail. The gradient colors in this project are also generated programmatically and can be modified to transition between any colors. Please feel free to modify the codes.

Keywords: graphic-design, procedural-generation, animated
