About This Work

Exquisite Corpse used three modular parts to create this figure. To create a number of vertex shapes that would be difficult to achieve manually in a short time, I developed a tool that supports creating polygonal shapes and exporting them as beginShape, vertex, and endShape formats in the console. With this tool, I was able to create fairly complex shapes in this work.

Keywords: procedural-generation, modular, sketch
